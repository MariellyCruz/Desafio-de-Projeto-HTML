História da Web

Primeiros computadores

Voltar

Em 1944 surgiu um computador eletromecânico usado para realização de cálculos complexos, chamado de Harvard Mark 1. Em 1946, meados da Segunda Guerra Mundial, surgiu o Colossus, criado pela Alemanha para crypto análises. E na mesma época, nos Estados Unidos, surgiu o ENIAC, que funcionava de forma automática. Nessa época, também, o termo "bug" foi trazido para a informática.

Na segunda geração de computadores, de 1959 a 1965, o uso de transistores ajudou a diminuir o tamanho dos computadores. A terceira geração, de 1965 a 1970, foi marcada pelo uso dos chips (circuito integrado) que diminuíram ainda mais o tamanho dos computadores e aumentou a capacidade dos mesmos.

A quarta geração, a partir de 1971, foi marcada pelo surgimento dos PCs (personal computers), com Bill Gates (Microsoft) e Steve Jobs (Apple) que popularizaram e acessibilizaram o uso dos computadores. A quinta geração se daria por computadores quânticos, crypto moedas, metaverso e pelas inteligências artificiais.

Surgimento da WEB

Voltar

A Web se desenvolveu paralelamente ao desenvolvimento dos computadores. As primeiras comunicações remotas foram realizadas em 1958 pelo telégrafo, usando o código morse. Um cabo de ponta a ponta foi colocado permitindo essa forma de comunicação.

Em 1950, período da Guerra Fria e com corrida espacial, houve um grande desenvolvimento das tecnologias. Um exemplo é o computador ENIAC, criado para realizar o cálculo de rotas de mísseis, baseado em diversas variáveis.

A DARPA - agência de pesquisa e defesa - surgiu e, em 1969, a ARPANET – sistema de comunicação em pacotes, múltiplos destinos – fez sua primeira conexão e conseguiu enviar uma mensagem.

Posteriormente, a ARPANET ganhou vários pontos e teve a implantação do sistema TCP/IP. Com o tempo, houve o surgimento de outras redes mais especializadas, como a MILNET que era exclusivamente militar.

Joseph Licklider, um cientista estadunidense, teorizou sobre uma rede global de computadores onde todos poderiam se comunicar. Em 1989, Tim Berners-Lee criou o primeiro navegador WorldWideWeb, baseado em hipertexto.

O que são Clients?

Voltar

É um termo usado para designar o lado cliente, ou seja, aquele dispositivo que consome as informações da internet. Após o cliente realizar a requisição, o servidor retorna arquivos estáticos para o cliente, como por exemplo arquivos HTML, CSS, JAVASCRIPT e imagens. Esse tráfego de informações depende diretamente da velocidade da conexão com a internet.

O navegador grava na máquina cliente arquivos “em cache”, onde algumas informações ficam gravadas para agilizar essa comunicação, não precisando fazer o download delas novamente num acesso futuro.

Entender o lado do cliente é importante para quem trabalha ou deseja trabalhar com desenvolvimento Front-end, pois é ele quem vai receber as informações. Portanto, deve-se pensar em compatibilidade entre navegadores e estilos de resoluções, por exemplo.

Os navegadores são programas instalados no dispositivo e servem para interpretar arquivos que vêm do servidor. Eles estão cada vez agregando mais recursos em sua composição e, para manter um padrão entre eles, há o W3C. As aplicações web são soluções criadas que possuem a internet como meio de comunicação cliente x servidor, mas não necessitam de instalação, como o navegador. O Facebook, por exemplo, se for acessado pelo navegador é uma aplicação web, mas quando ele é acessado pelo aplicativo instalado no dispositivo é um aplicativo.

Atualmente, é extremamente importante pensar em aplicabilidade para dispositivos móveis, pois, segundo pesquisas, nove em cada dez latino-americanos possuem ou usam um dispositivo móvel.

O que são Servers?

Voltar

Servidores são computadores que servem de apoio para outras máquinas ou sistemas e podem ou não ser acessíveis pela internet, uma vez que existem servidores internos que formam uma rede local. Eles são mais robustos, porque exigem processamentos e devem servir a muitos dispositivos. Um exemplo de servidores são os data centers.

Existem os servidores de arquivos, os de segurança, de streaming, de e-mail e web. Como servidores de segurança, por exemplo, tem-se o proxy e o firewall, onde o primeiro bloqueia acesso do cliente e o segundo bloqueia acessos externos.

No computador, o hardware é a parte física, com função de armazenamento, memória e processadores. A parte lógica, chamada software consiste em sistemas operacionais, monitoramento e servidores web.

O que é a Linguagem de Programação?

A linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software). As 3 linguagens mais usadas atualmente em 2022 são: Javascript, Phyton e Java.

Linguagens server-side são as linguagens executadas no servidor, como por exemplo o PHP, Ruby, C#, Java e Javascript, que possuem uma sintaxe específica. As linguagens client-side são as que são executadas pelo usuário final, como Javascript. O Javascript possui um módulo dentro do navegador e do servidor chamado V8 para que ele seja executado e interpretado

Introdução ao HTML

Voltar

HTML não é uma linguagem de programação, mas sim um arquivo de texto com delimitadores que são interpretados pelo navegador. Ele possui três partes: os elementos (TAGs) para abrir e fechar o HTML; dentro dele estão o cabeçalho (head) e o corpo (body). No Head são inseridas coisas que o arquivo vai carregar antes de abrir para o usuário e no Body é o que de fato vai aparecer para o usuário.

O que são TAGs e seus atributos?

Voltar

São elementos usados para informar ao navegador a estrutura do site. Os atributos das tags são algumas propriedades que elas podem ter. Na tag "strong", por exemplo, podemos usar o atributo "id" para identificar o elemento que será alterado.

No HTML ainda é possível realizar mudanças no tamanho dos títulos e subtítulos, criar listas ordenadas e não ordenadas e criar links dentro do site.
